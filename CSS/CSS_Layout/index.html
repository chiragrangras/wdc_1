All modern website and web applications are built using three fundamentaltechnologeis:HTML,CSS and JavaScript.These are the languages of the web.

What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that web developers use to structure and descibe the contect of a webpage (not a programming language).

HTML consits of elements that describe different type of contect: paragrahs,links,headings,images,video,etc. Web browsers understand HTML and render HTML code as websites.

In HTML each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDV Web Docs.
